Sobre mim

Sou Paulo Rezende, Desenvolvedor Full Stack Jr com background comercial e experiência prática no desenvolvimento de aplicações web, APIs e dashboards voltados para operação, produtividade e tomada de decisão.

Minha transição para tecnologia trouxe uma visão forte de negócio, comunicação e foco em resultado. Hoje atuo com desenvolvimento front-end e back-end utilizando JavaScript, TypeScript, React, Node.js, Python, MySQL, PostgreSQL e Prisma, além de integração entre sistemas, modelagem de banco de dados e consumo e criação de APIs REST.

Também tenho vivência com deploy e infraestrutura em AWS, uso de Git e GitHub no versionamento e construção de interfaces responsivas a partir de layouts e necessidades reais do negócio. Gosto de transformar problemas em soluções úteis, com código organizado, manutenção simples e foco na experiência do usuário.

JavaScript

Git

GitHub

Node.js

Trabalho com Node.js na criação de APIs REST, regras de negócio, integração com banco de dados e comunicação entre serviços. É uma tecnologia que utilizo em aplicações web completas, principalmente quando preciso construir back-ends leves, rápidos e organizados.

SQL & APIs

Tenho experiência com consultas SQL e com consumo e criação de APIs REST para integrar sistemas, alimentar dashboards e automatizar fluxos. Consigo trabalhar com manipulação de dados, filtros, relacionamentos e envio de informações entre front-end, back-end e banco de dados.

React

Utilizo React para desenvolver interfaces componentizadas, dinâmicas e reutilizáveis, facilitando a organização do front-end e a escalabilidade da aplicação. Gosto de estruturar componentes de forma clara para melhorar manutenção, leitura do código e experiência do usuário.

MySQL

Uso MySQL em sistemas que exigem armazenamento estruturado, consultas relacionais e controle consistente dos dados. Tenho prática com criação de tabelas, relacionamentos, filtros, consultas e integração com aplicações para suportar funcionalidades reais de negócio.

PostgreSQL

Também trabalho com PostgreSQL em aplicações que pedem robustez, integridade e boa modelagem relacional. Utilizo essa tecnologia em conjunto com back-end e ORM para estruturar dados de forma confiável e manter aplicações mais preparadas para crescimento.

AWS

Tenho vivência com serviços da AWS para deploy e infraestrutura, incluindo hospedagem de aplicações, banco de dados e armazenamento. Isso me permite participar não apenas do desenvolvimento do sistema, mas também da parte de publicação e sustentação em ambiente cloud.

TypeScript

Uso TypeScript para tornar o código mais seguro, previsível e fácil de manter, especialmente em aplicações com maior volume de regras e componentes. A tipagem ajuda a reduzir erros, melhorar a legibilidade e dar mais confiança durante a evolução do projeto.

Python

Utilizo Python em automações, manipulação de dados e projetos voltados à lógica aplicada, incluindo cenários com visão computacional. É uma linguagem que me ajuda a resolver problemas com rapidez e clareza, principalmente em soluções técnicas fora do escopo tradicional do front-end.

Prisma

Uso Prisma como ORM para facilitar a comunicação entre back-end e banco de dados, com consultas mais organizadas, tipadas e seguras. Ele contribui bastante para produtividade, clareza na modelagem e manutenção do código em projetos com Node.js e TypeScript.

NestJS

Tenho conhecimento em NestJS para estruturar back-ends de forma mais modular, escalável e padronizada. A proposta do framework é útil para aplicações maiores, com separação clara de responsabilidades, controllers, services e organização mais próxima de arquitetura corporativa.

Express.js

Utilizo Express.js na criação de APIs e rotas para aplicações Node.js, principalmente em projetos que exigem agilidade, simplicidade e controle direto da estrutura do back-end. É uma ferramenta importante na construção de serviços web leves e funcionais.

Nginx

Tenho experiência com Nginx na configuração de servidores, proxy reverso e publicação de aplicações. Seu uso é importante para melhorar entrega, organizar acesso aos serviços e apoiar ambientes de produção com mais estabilidade e controle.

PM2

Utilizo PM2 para gerenciar processos Node.js em ambiente de produção, mantendo aplicações ativas, monitoradas e com reinício facilitado quando necessário. Isso ajuda bastante na sustentação e disponibilidade de sistemas publicados.

Deploy

Participo do processo de deploy de aplicações, desde a preparação do ambiente até a publicação e validação do funcionamento em produção. Tenho familiaridade com ajustes de servidor, configuração de variáveis, execução de serviços e acompanhamento pós-publicação.

Responsividade

Desenvolvo interfaces responsivas pensando em adaptação real para diferentes tamanhos de tela, garantindo boa usabilidade em desktop, tablet e celular. Busco equilíbrio entre layout, legibilidade, espaçamento e navegação em qualquer dispositivo.

Figma

Consigo interpretar layouts e protótipos feitos no Figma para transformar design em interface funcional, respeitando hierarquia visual, espaçamentos, componentes e identidade do projeto. Isso facilita a comunicação entre design e desenvolvimento.

Visão Computacional

Tenho experiência prática em projetos com visão computacional, utilizando OpenCV para captura, tratamento e análise de imagens em contextos industriais. É uma área em que aplico lógica, processamento visual e automação para resolver problemas reais.

Dashboards

Desenvolvo dashboards voltados para operação e tomada de decisão, com indicadores, filtros, visualização clara dos dados e integração com banco de dados. Gosto de transformar informação em algo visualmente útil para acompanhamento de processos e resultados.

Arquitetura

Tenho atenção à organização da aplicação, separação de responsabilidades e estruturação do código para facilitar manutenção, evolução e entendimento do sistema. Busco aplicar boas práticas de arquitetura em camadas e modularização sempre que o projeto exige mais clareza e escala.
